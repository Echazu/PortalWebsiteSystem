## 📋 About PortalWebsiteSystem

PortalWebsiteSystem is a company portal designed to help you manage your website easily. It is built using Springboot and Thymeleaf. The system uses layui for a clean and user-friendly interface, with parts of the website running separately from the server.

This system works with common tools like MySQL for storing your data, Redis for quick access to information, and Shiro for security. You do not need to know programming to use PortalWebsiteSystem.

## 💻 System Requirements

Make sure your computer meets these needs before installing:

## 🔧 How to Use PortalWebsiteSystem

PortalWebsiteSystem shows a clean dashboard after you open it. Here’s how to get started:

- **Login Screen:** Use the default username and password provided in the user manual or contact your system administrator.  
- **Homepage:** From here, you can see your company news, quick links, and settings.  
- **Menus:** On the left, find menus such as "Company Info," "News," "User Management," and "Settings." Click any section to open it.  
- **Editing Content:** To add or change information, use the "Edit" buttons on each page. You do not need to write code; just type or upload files.

## 🗄️ Data and Storage

PortalWebsiteSystem uses MySQL to store your company data securely. You do not need to manage this database manually because the software handles it behind the scenes.

Redis is used by the system to speed up access to common data, making the portal faster.

---

## 🔒 Security Features

- **User Accounts:** Each user gets a secure login with their own password.  
- **Permissions:** Access to some parts of the portal can be limited based on user roles.  
- **Encryption:** Sensitive data is encrypted during use.  
- **Session Control:** The system automatically logs out users who stay inactive too long.
